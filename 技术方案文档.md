## 买家说 — 技术方案文档

### 一、项目概述

「买家说」是一个购物种草社区 H5 应用，用户可以发布购物体验、浏览好物推荐、互动评论点赞。面向移动端优先设计，部署到 EdgeOne Pages。

### 二、技术栈

| 层面 | 选型 | 版本 | 说明 |
|------|------|------|------|
| 框架 | Next.js (App Router) | 14.x | React 元框架，支持 SSG/SSR |
| 语言 | TypeScript | 5.4 | 类型安全 |
| 样式 | Tailwind CSS | 3.4 | 原子化 CSS |
| 动画 | Framer Motion | 11.x | 页面转场、微交互 |
| 状态 | Zustand | 4.5 | 轻量状态管理 (<1KB) |
| 后端 | Supabase | - | PostgreSQL + Auth + Storage（待接入） |
| 部署 | EdgeOne Pages | - | 静态导出，腾讯 CDN |

### 四、页面设计

| 页面 | 路由 | 核心功能 |
|------|------|----------|
| 首页/发现 | `/` | 标签筛选 + 瀑布流双列 + 骨架屏 + 下拉刷新 + 触底加载 |
| 帖子详情 | `/post/[id]` | 图片轮播 + 商品信息 + 评分 + 点赞心跳动画 + 评论 |
| 发布 | `/publish` | 多图上传 + 表单 + 评分 + 标签选择 + 登录检查 |
| 搜索 | `/search` | 搜索框 + 热门标签 + 搜索历史 + 结果列表 |
| 个人中心 | `/profile` | 用户信息 + 数据统计 + 我的帖子 + 菜单 |

### 五、UI 设计规范

**色彩系统**：
- 主色：珊瑚红 `#FF6B35`（coral-500）
- 背景：暖白 `#FAF7F5`（warm-50）
- 强调：渐变 `from-coral-500 to-coral-400`

**交互规范**：
- 卡片：圆角 16px + 微阴影 + 按压缩放 (scale 0.98)
- 按钮：圆角胶囊 + 渐变色 + 按压反馈
- 列表：错落淡入动画 (staggerChildren, 60ms 间隔)
- 页面转场：Framer Motion fadeInUp
- 点赞：心跳动画 (scale + rotate 组合)
- 加载：骨架屏 shimmer 动画

**移动端**：
- 最大宽度 430px 居中
- 底部安全区 (safe-area-inset-bottom)
- 触摸反馈 (active:scale)

### 六、数据库设计（Supabase）

```sql
-- 用户资料
CREATE TABLE profiles (
  id UUID PRIMARY KEY REFERENCES auth.users(id),
  nickname TEXT NOT NULL,
  avatar_url TEXT DEFAULT '',
  bio TEXT DEFAULT '',
  post_count INT DEFAULT 0,
  follower_count INT DEFAULT 0,
  following_count INT DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 帖子
CREATE TABLE posts (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES profiles(id),
  title TEXT NOT NULL,
  content TEXT NOT NULL,
  images TEXT[] DEFAULT '{}',
  tags TEXT[] DEFAULT '{}',
  product_name TEXT DEFAULT '',
  price TEXT DEFAULT '',
  rating INT DEFAULT 5,
  like_count INT DEFAULT 0,
  comment_count INT DEFAULT 0,
  status TEXT DEFAULT 'active',
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 评论
CREATE TABLE comments (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  post_id UUID REFERENCES posts(id) ON DELETE CASCADE,
  user_id UUID REFERENCES profiles(id),
  content TEXT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 点赞
CREATE TABLE likes (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  post_id UUID REFERENCES posts(id) ON DELETE CASCADE,
  user_id UUID REFERENCES profiles(id),
  created_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(post_id, user_id)
);
```

### 七、部署流程

1. 推送到 GitHub
2. EdgeOne Pages 自动拉取，执行 `edgeone.json` 中的构建命令
3. `npm install` → `npm run build` → 输出到 `out/` 目录
4. 腾讯 CDN 分发静态资源

```json
{
  "buildCommand": "npm run build",
  "installCommand": "npm install",
  "outputDirectory": "out",
  "nodeVersion": "20.18.0"
}
```
