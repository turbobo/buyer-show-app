# 买家说 — 产品设计文档

> 购物种草社区 H5 应用，让用户分享真实购物体验，发现好物推荐。
> 版本：v0.3 | 更新：2026-06-22

---

## 一、产品定位

**一句话描述**：真实买家的购物分享社区，帮你买到真正值得的好物。

**目标用户**：
- 18-35 岁年轻消费者，习惯网购
- 喜欢看真实用户评价再做购买决策
- 乐于分享自己的购物心得和使用体验

**核心价值**：
- 对读者：看到真实、可信的购物评价，避免踩坑
- 对作者：记录购物体验，获得社区认同
- 对品牌：获取真实用户反馈和口碑传播

## 七、技术架构

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   用户浏览器  │────▶│  EdgeOne Pages   │────▶│   Supabase      │
│  (手机/PC)   │     │  腾讯 CDN 静态托管 │     │  PostgreSQL     │
└─────────────┘     └──────────────────┘     │  Auth           │
                                              │  Storage        │
                                              │  Realtime       │
                                              └─────────────────┘
```

- **前端**：Next.js 14 (App Router) + React 18 + TypeScript + Tailwind CSS + Framer Motion
- **状态**：Zustand (user store + UI store)
- **后端**：Supabase (PostgreSQL + Auth + Storage + Edge Functions)
- **部署**：EdgeOne Pages 静态导出，腾讯 CDN 全球分发
- **域名**：待绑定
