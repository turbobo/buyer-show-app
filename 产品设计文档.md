# 买家说 — 产品设计文档

> 购物种草社区 H5 应用，让用户分享真实购物体验，发现好物推荐。
> 版本：v0.8 | 更新：2026-06-24

---

## 一、产品定位

**一句话描述**：真实买家的购物分享社区，帮你买到真正值得的好物。

**目标用户**：
- 18-35 岁年轻消费者，习惯网购
- 喜欢看真实用户评价再做购买决策
- 乐于分享自己的购物心得和使用体验

**核心价值**：
- 对读者：看到真实、可信的购物评价，避免踩坑
- 对作者：记录购物体验，获得社区认同
- 对品牌：获取真实用户反馈和口碑传播

## 七、技术架构

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   用户浏览器  │────▶│  EdgeOne Pages   │────▶│   Supabase      │

                                              │  Realtime       │
                                              └─────────────────┘
```

- **部署**：EdgeOne Pages 静态导出，腾讯 CDN 全球分发

### 评论功能设计建议（5.4）

**已实现**：
- 评论删除（作者可删，二次确认，级联删除子回复）
- 一级嵌套回复（`parent_id` 外键，只允许回复顶级评论，不支持回复的回复）
- 桌面端评论输入（评论区底部内联输入框，移动端保留固定底栏）
- 评论收藏（收藏按钮 + `favorite_comments` 表）

**推荐后续迭代**：
- 评论点赞：独立 `comment_likes` 表 + 触发器维护 `like_count`，与收藏区分
- 评论分页：评论超过 20 条时分页加载（"查看更多评论"按钮）
- 评论排序：支持「最新」/「最热」切换（按 `created_at` / `favorite_count` 排序）
- 评论举报：举报按钮 + `comment_reports` 表 + 管理后台审核
- 评论编辑：作者可编辑（发布后 5 分钟内），标记「已编辑」
